Fundamentals of JavaScript 1

Цыганков Артем Б21-525

Задание 1

Сделать программу, которая принимает на вход номер месяца от 1 до 12 и возвращает название месяца. Если месяц не существует, то вернуть “Некорректный номер месяца”.

Задание 2

Сделать программу, которая принимает на вход натуральное число n и возвращает список из n простых чисел, разделенных пробелом.

Задание 3

Сделать объект Counter, обладающий свойством count и методами add(value) и sub(value). При вызове этих методов свойство count должно увеличиваться или уменьшаться на величину value, соответственно.

Задание 4

Сделайте программу, которая принимает на вход список слов, разделенных запятыми и возвращает этот же список слов, разделенных точками.

Задание 5

Сделайте программу, которая принимает на вход строку и определяет, является ли строка палиндромом. Программа должна вернуть “Да” или “Нет”
